# Anime1 番剧更新列表

AstrBot 插件，用于获取 [anime1.me](https://anime1.me) 的番剧更新列表，并提供 AI 工具接口。

## 功能

- **定时更新**：每天指定时段自动获取番剧列表，去重合并后保存
- **AI 工具 `get_anime_list`**：让 AI 获取番剧列表，支持缓存开关、时间范围过滤和数量限制
- **AI 工具 `get_watch_url`**：让 AI 通过番剧 ID 获取 anime1 观看地址
- **手动更新指令**：`/anime_update` 手动触发更新

## 安装

在 AstrBot 管理面板的插件页面，上传插件 zip 包或填写仓库地址安装。

## 配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `update_times` | 每日更新时段（小时，0-23，逗号分隔） | `1` |

示例：
- `1` - 每天凌晨 1 点更新
- `1,13,22` - 每天 1:00、13:00、22:00 更新

## AI 工具说明

### get_anime_list

获取番剧更新列表。

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `use_cache` | boolean | 否 | 是否使用缓存，默认 true |
| `time_range` | string | 否 | 时间范围：`年`、`月`、`周`、`日`、空，默认 `日` |
| `limit` | number | 否 | 返回数量限制，默认 100，-1 表示无限制 |

**返回示例：**
```
共 500 部番剧，返回 3 部：
[1868] 異世界悠閒農家2(第二季) - 連載中(01) (2026年春)
[1867] 木頭風紀委員和迷你裙JK的故事 - 連載中(01) (2026年春)
[1832] 卡片戰鬥!! 先導者 Divinez 第五季 幻真星戰篇 - 連載中(10) (2026年冬)
```

### get_watch_url

通过 ID 获取 anime1 观看地址。

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `anime_id` | number | 是 | 番剧 ID |

**返回示例：**
```
观看链接：https://anime1.me/xxxxx
```

## 数据存储

番剧数据存储在 `data/plugin_data/astrbot_plugin_anime1_list/anime_list.json`。
